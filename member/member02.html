氏名：
古橋大地
経歴：
2001年 東京大学大学院 新領域創成科学研究科 環境学専攻 修了
2001年〜2005年 株式会社オープンGIS
2005年〜 東京大学空間情報科学研究センター 特任研究員（現職）
2005年〜 マップコンシェルジュ株式会社 代表取締役（現職）

専門は森林リモートセンシング。
地理空間情報の利活用を軸に、Googleジオサービス、オープンソースGIS(FOSS4G)、オープンデータ(OpenStreetMap)の技術コンサルティングや教育指導を行なっている。ここ数年は「一億総伊能化」をキーワードにみんなで世界地図をつくるOpenStreetMapに熱を上げ、GPS、パノラマデバイス(GigaPan)、無人航空機(UAV/Drone)を駆使して、地図を作るためにフィールドを駆け巡っている。最近の主なプロジェクト：・JICA Project /ブラジル国アマゾン森林保全・違法伐採防止のためのALOS衛星画像の利用プロジェクト(Alos4Amazon:http://alos4amazon.com/)・世界防災減災ハッカソン Race for Resilience / Code for Resiliencehttp://raceforresilience.org/現在日本のクライシスマッピングチーム立ち上げに奔走中。
今回のDECOで採用したすごい災害対応訓練は、Race for Resilienceの成果の一つ。

現場を味方にしろ！：その土地の地誌を理解し、ホームグラウンドにする。

DIG（図上訓練）という手法を使って、浦安という土地の成り立ちを解説、直下型地震が発生した時のシナリオを体験する。すべての作業では「地図」を用いて理解し、その情報を現場で持ち歩くためのテクニックを学びます。

・浦安という土地の成り立ちの理解
・地図が読めるようになる
・地図を現場で持ち歩くためのテクニックを学ぶ

結果として非常時の対応で、自分たちがどこにいて、何をしたらいいのか、どこに行ったらいいのかを考えられるようになってほしい。

今までの画一的な避難訓練から、より地域に適応した「楽しめる」災害訓練へと進化する、そんなDECOの取り組みに期待し、積極的に参画しています。
